# 📝 To-Do List Application

A simple command-line To-Do List application built with Python. This project allows users to add tasks, view their tasks, and manage daily activities through a user-friendly menu-driven interface.

## 🚀 Features

* Add new tasks
* View all saved tasks
* Simple and easy-to-use command-line interface
* Continuous menu loop until the user chooses to exit

## 📂 Project Structure

```
To-Do-List/
│
├── todo.py
└── README.md
```

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open a terminal in the project directory.
4. Run the program:

```bash
python todo.py
```

## 📖 Usage

After running the program, you will see the following menu:

```
===== TO-DO LIST =====
1. Add Task
2. View Tasks
3. Exit
```

### Add a Task

Select option `1` and enter your task.

### View Tasks

Select option `2` to display all added tasks.

### Exit Program

Select option `3` to close the application.
