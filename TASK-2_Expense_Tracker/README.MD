# 💰 Expense Tracker

A simple command-line Expense Tracker application built using Python. This project allows users to record expenses, calculate the total amount spent, and view a summary of their spending.

## 📌 Project Overview

The Expense Tracker helps users keep track of their daily expenses by continuously accepting expense amounts and calculating the total spending. It demonstrates fundamental Python concepts such as loops, variables, arithmetic operations, error handling, lists, and accumulators.

## 🚀 Features

* Add multiple expenses
* Automatically calculate total spending
* Display all recorded expenses
* Calculate average expense
* Find highest expense
* Find lowest expense
* Input validation using exception handling
* Exit using a sentinel value (`quit`)

## ▶️ How to Run

1. Install Python 3 on your system.
2. Download or clone this repository.
3. Open the terminal in the project folder.
4. Run the following command:

```bash
python expense_tracker.py
```

## 📖 Usage

Enter expense amounts when prompted.

Example:

```text
Enter expense amount (or 'quit'): 100
Current Total: ₹100

Enter expense amount (or 'quit'): 50
Current Total: ₹150

Enter expense amount (or 'quit'): 20
Current Total: ₹170

Enter expense amount (or 'quit'): quit
```
