# 🔐 Random Password Generator

A simple and secure Random Password Generator built using Python. This project generates strong passwords containing uppercase letters, lowercase letters, numbers, and special characters based on the user-defined password length.

## 📌 Project Overview

Passwords are the first line of defense for protecting online accounts and sensitive information. This application helps users create strong and random passwords that are difficult to guess or crack.

The project demonstrates the use of Python's built-in modules such as `random` and `string` for secure password generation.

## 🚀 Features

* Generate random passwords of any length
* Includes:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
  * Special characters (!, @, #, $, etc.)
* User-defined password length
* Input validation using exception handling
* Simple command-line interface

## 🛠️ Technologies Used
